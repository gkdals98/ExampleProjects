## Tirano Rush

+ webpack + html, vue, scss만 사용해 텍스트 파일 하나 던져주고 그걸 인터넷되는 환경(cdn)에서 html로 저장하기만 하면 바로 수행할 수 있는 게임을 만드는게 목표다.
+ html 파일 한 개로 10분짜리 커피타임 브레이커를 만든다.
+ 구글의 클릭으로 티라노 점프하는 그 미니게임처럼 우리는 오로지 클릭만으로 간단하게 진행하는 로그라이크를 만들자.

## 기획
+ 아주 심플하게 아래와 같은 흐름으로 가자.
1. 메뉴화면에서 start 누르면 인트로 없이 게임이 시작되며 루프 돌입
2. 가챠타임이 오고 가진 자원만큼 가챠를 돌려 리스트가 쭉 나온다. 원하는 캐릭터를 선택하면 자동으로 배치(최대 스테이지 수만큼의 캐릭터 보유)되고 스테이지 시작
3. 스테이지를 계산해 적정 수준의 몹(스테이지 수만큼)이 등장한다.
4. 그냥 클릭하면 8bit 노이즈와 함께 바로 전투 결과가 바로 나온다.
5. 몹을 쓰러뜨리면 아이템이 세 개 출현하며 이 중 하나를 선택할 수 있다.
6. 스테이지에 지정된 횟수를 반복하면 루프 처음의 가챠타임으로.

+ 이런 기획은 보통 게임 자체는 더럽게 재미없기때문에 상황설정으로 상상력을 불어넣어줘야한다. 집중력을 높이는데 좋은건 가챠의 재미를 올려줄 캐릭터, 단지 스텟만 올려주는게 아니라 그럴싸한 설정이 붙은 아이템, 적 몬스터들의 통일된 분위기 등이다. 텍스트로만 만들거니 뭔가 상상력을 자극하는 문장으로 작명과 설명을 써넣는 것으로 이를 구현한다.
